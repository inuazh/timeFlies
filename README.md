# timeFlies

A life calendar that visualizes your 75 years as a grid of weeks. See the past, the present, and what's left.

**[timeflies-production.up.railway.app](https://timeflies-production.up.railway.app)**


- Set goals on future weeks - color-coded tracks show your timeline
- Right-click a goal to delete it
- Export to PDF

## Stack

- Next.js 16 · React 19 · TypeScript
- Redux Toolkit
- SQLite 
- JWT authentication
- CSS Modules · glassmorphism UI
- Feature-Sliced Design

## for local setup

```bash
npm install
cp .env.local.example .env.local  # add your JWT_SECRET
npm run dev
```
